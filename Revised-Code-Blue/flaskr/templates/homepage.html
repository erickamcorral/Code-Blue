Acting Fast

Strokes can be unexpected and very dangerous, which is why it is important to take immediate action when stroke symptoms are identified. This is where FAST can help. By scanning facial images, FAST detects potentially dangerous facial asymmetry and notifies an emergency contact.

About FAST

FAST is a web application for stroke detection developed by Code Blue. FAST utilizes artificial intelligence to recognize facial paralysis. Users can acess FAST on any device with an internet connection.
